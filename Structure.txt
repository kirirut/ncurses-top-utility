PID	ID процесса
USER	Владелец процесса
PR	Приоритет процесса
NI	Nice (низкий = приоритет выше)
VIRT	Виртуальная память (включая swap)
RES	Физическая RAM, занятная процессом
SHR	Разделяемая память (shared memory)
S	Статус процесса: R (running), S (sleeping), Z (zombie)
%CPU	Процент загрузки CPU
%MEM	Процент использования RAM
TIME+	Общее время работы процесса
COMMAND	Имя исполняемого файла
